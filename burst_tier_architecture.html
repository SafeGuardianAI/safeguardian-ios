Adaptive Transport Logic

  • Spatial awareness (UWB): Nova 1 uses NearbyInteraction to measure the exact distance to Nova 2. This metadata is used to optimize the BLE routing path without sending extra data.
  • High-Speed Burst (MPC): When Nova 2 needs to send a large structural photo, it triggers a MultipeerConnectivity link to the Trek node. This provides Mbps throughput, bypassing the 15KB/s BLE limit.
  • Unified P2P Host: All nodes run a libp2p-compatible substrate, allowing them to switch between BLE, UWB, and MPC transports seamlessly based on the payload size and distance.
